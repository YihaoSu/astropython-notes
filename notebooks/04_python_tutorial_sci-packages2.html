四堂課程大綱¶

  • 第一堂-Python 基礎(一)：Python 簡介及環境建立、Python程式的編寫及執行、資料型態、基本輸入輸出、流程控制

Scipy提供以下模組:¶

  • 特殊函數 (scipy.special)
  • 數值積分/解微分方程 (scipy.integrate)
  • 最佳化/擬合/求根 (scipy.optimize)
  • 內插 (scipy.interpolate)
  • Fourier 轉換 (scipy.fftpack)
  • 訊號處理 (scipy.signal)
  • 線性代數 (scipy.linalg)
  • 機率分佈/統計函數 (scipy.stats)
  • 多維影像處理 (scipy.ndimage)
  • ...

def lc_model(time, amplitude, period, dc):
    return amplitude * np.sin(2 * np.pi * 1. / period * time) + dc
In [4]:
popt, pcov = curve_fit(lc_model, time, counts,  p0=[50, 2.5, 100], sigma=counts_err)
print(popt)
print(pcov)

Astropy提供以下模組:¶

  • 物理常數及單位換算 (astropy.constants, astropy.units)

  • 天球座標系統轉換 (astropy.coordinates)
  • 資料擬合 (astropy.modeling)
  • FITS檔操作 (astropy.io.fits)
  • ...
